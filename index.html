Automobile Fuel Efficiency

Project Data

Data used in this project contains average data for 2017 automobile manufacturers. It can be downloaded from https://flunky.github.io/cars2017.csv. Visualization provides a good insight for fuel effiency.

1. Hybrid Structure

The Visualization follows the Martini Glass hybrid structure. Initial 3 scenses provides insight for different kind of fuels and the last one contains all data.

2. Scenes

Consecutive scenes reflecting the effeciency per fuel in order - Gasoline, Diesel, Electricity. Last scene has no filters and contains all data.

3. Annotations

Paragraphs and tool tips provides the Annotations.

4. Parameters

Data and Fuel type serves as the parameter.

5. Triggers

Slide show with forward and backword buttons are the triggers.
